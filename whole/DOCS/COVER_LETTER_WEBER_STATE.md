## The Core Idea

Traditional computing assumes data must be stored to be useful. ButterflyFX challenges this assumption by demonstrating that when data can be deterministically derived from a well-defined mathematical structure, storing it as redundant records becomes unnecessary.

The framework stores only a minimal "generative basis"—code, manifold definitions, and interpretation rules—from which application data is derived on demand through geometric evaluation.

## Why This Matters

If viable at scale, storage-free computing could:

- **Reduce data redundancy** across systems
- **Eliminate synchronization conflicts** inherent in distributed data
- **Enable deterministic, verifiable computation** with strong audit guarantees
- **Simplify system architecture** by removing persistence complexity

## Mathematical Foundation

The framework is built on established mathematics:

- **The Schwarz Diamond minimal surface** (discovered 1865)
- **Helix geometry** with constant curvature and torsion
- **The primitive interaction `z = x·y`** defining a hyperbolic paraboloid

These are not speculative constructs—they are well-understood mathematical objects applied in a novel computational context.

## Empirical Verification

**All 43 tests pass.** The tests are reproducible and available in the open-source repository.
